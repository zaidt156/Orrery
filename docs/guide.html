Using Orrery

Your private AI workspace, made simple.

This guide walks you through installing Orrery and using it day to day - in plain language, no jargon. If you can install an app and paste a key, you're ready.

01What is Orrery

Orrery is a desktop app that brings AI into one private workspace on your own computer. You connect the AI models you already use (Claude, ChatGPT, Google, or a free local model) and, optionally, your own database - and Orrery ties them together.

The important part: your files, chats, and data stay on your machine. Nothing is sent anywhere except to the AI provider you chose, and only when you ask it to.

ChatTalk to any model, with your documents in the loop.
Data & dashboardsAsk questions of your database; save live dashboards.
AgentsGive a goal and let a bounded worker do it.
Media & filesGenerate documents, images, and more.

02Install it

Windows

  1. Download the Windows installer (.exe) from the download page.
  2. Run it. If Windows shows a blue "protected your PC" box, click More info -> Run anyway (the app isn't code-signed yet - that's expected for a preview).
  3. Orrery installs like any app, with a Start Menu and desktop shortcut.

macOS

  1. Download the .dmg for your Mac - Apple Silicon (M1/M2/M3) or Intel.
  2. Open it and drag Orrery into your Applications folder.
  3. The first time, right-click Orrery -> Open (instead of double-clicking) to get past Gatekeeper. You only do this once.

03First run

Orrery keeps everything in a small database on your computer, and it sets that up for you automatically.

  • If you have Docker Desktop installed: just open Orrery. It starts Docker if needed, creates its database the first time, and opens - no setup screens.
  • If you don't have Docker: Orrery shows a friendly dialog with a button to get Docker Desktop. Install it, reopen Orrery, and you're set.
  • Already run your own PostgreSQL? You can point Orrery at it instead in Settings, and Docker isn't needed at all.
Tip: The very first launch asks you a couple of quick questions (what to call you, what you'll use Orrery for). Your answers become Orrery's private memory - you can edit or skip them any time in Settings.

04Connect a model

Orrery doesn't ship with an AI - you bring your own, which keeps you in control of cost and privacy. Open Settings -> Accounts and pick whatever you have:

  • An API key from OpenAI, Anthropic (Claude), Google, Mistral, DeepSeek, or any OpenAI-compatible service - paste it and you're done. Keys are stored in your operating system's secure keychain, never in a file.
  • A plan account - connect your Claude or ChatGPT subscription through their official app.
  • A free local model with Ollama - runs entirely on your machine, no key, no internet.

Then in Settings -> Models, turn on the ones you want to see in the chat menu. Switch models any time, even mid-conversation.

05Chat & your data

The Chat tab is the heart of Orrery. Type a question and the answer streams back. You can attach files (PDF, Word, Excel, images, code) and ask about them - Orrery reads them on your machine.

Use my data

Flip on Use my data and pick a collection to have Orrery answer from your own documents, citing which file each fact came from. Build collections in the Data and Ontology tabs by dropping in files.

Good to know: Orrery figures out what you actually want. Ask a calculation and you get an answer; ask for "a PDF report" and you get a real file. It won't turn a plain question into a random file.

06Projects

A Project is a workspace for related chats, with its own files and standing instructions. Create one for a client, a topic, or a piece of work - every chat inside it automatically knows that project's context, so you don't repeat yourself.

07Agents

Agents are workers you give a goal instead of a single message. You decide exactly what each agent can touch - which tools, which data - and set limits (how many steps, how much it can spend per day) and a schedule.

  • Press Run and watch every step, tool call, and result in a live feed.
  • Anything risky pauses and asks you before it happens - nothing acts outside its lane.
  • Schedule an agent to run on its own (say, every morning).

08Dashboards

Describe the dashboard you want in plain words and pick a model. Orrery reads your connected data, writes the queries, and picks the charts - once. After that it just refreshes on live data, so reopening it costs nothing. Every query is visible and read-only, so it can never change your data.

09Themes

Make Orrery yours in Settings -> Appearance. Choose from five color looks - Simple, Futuristic, Winter, Summer, and Observatory - and two layouts (a compact Classic view or a fuller Concept dashboard). Mix and match; the whole app restyles instantly.

10Your privacy

Privacy isn't a setting in Orrery - it's the design.

  • Your files, chats, database, and generated files stay on your computer.
  • Orrery talks only to the AI providers you set up - nothing else.
  • When you use a cloud model, Orrery can automatically mask personal details (emails, phone numbers, card numbers) before anything is sent. Use a local model and nothing leaves at all.
  • No analytics, no tracking, no "phone home".

11Troubleshooting

The app won't open / "setup failed"

This almost always means the database isn't up yet. Make sure Docker Desktop is installed and let it finish starting, then reopen Orrery - it brings its database up automatically. If Docker isn't installed, the app shows a button to get it.

Windows "protected your PC" / macOS "damaged" or "unidentified developer"

Preview builds aren't code-signed yet. On Windows click More info -> Run anyway; on macOS right-click -> Open the first time. A signed release is planned.

A model says it's out of credit / hit a limit

That's your provider account, not Orrery. Top up with that provider, or switch to another connected model (or a local Ollama model) in the chat menu.
