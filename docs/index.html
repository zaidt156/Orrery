Orrery

A private AI workspace that connects chat, data, dashboards, automations, agents, and media generation to the models and databases you already choose.

Download WindowsView source
Private by defaultYour keysYour database
One workspace

Every surface shares the same context.

Orrery is not a set of disconnected demos. Chat can inspect data, dashboard specs can be saved and refreshed, automations can call agents, and agents can hand work back to you.

Chat

Stream from connected local or cloud models with retrieval over your documents.

Data

Connect PostgreSQL, browse schemas, and build pgvector-backed collections.

Dashboards

Ask for a dashboard and save the generated SQL, chart, and layout spec.

Automations

Run fixed workflows from a schedule, webhook, database change, or manual trigger.

Agents

Goal-driven workers plan, act, check results, and keep learning notes.

Media

Generate image and video assets from your own media keys or local model setup.

Security model

Private by default, explicit by design.

Orrery runs on localhost, stores secrets in the operating system keychain, and treats model-written actions as something to constrain and review.

The app contacts only the services you configure, so useful AI stays close to your data without becoming an opaque cloud handoff.

Keychain secrets

Provider keys are stored through the OS credential store.

Sandboxed tools

Tool execution is gated behind explicit local scopes.

Visible SQL

Database work is generated as inspectable SQL.

No phone-home

No analytics pipeline is required for the app to run.

Start

Four steps from download to workspace.

The setup path is built around local prerequisites: PostgreSQL, optional Docker, and whichever model providers you want to connect.

Extract or install

Use the installer, or keep the portable folder together and run setup-orrery.bat.

Choose a database

Use the bundled Docker PostgreSQL path or paste your own connection string.

Connect models

Add provider keys or point Orrery at local models. Secrets stay in the OS keychain.

Start building

Chat, create a dashboard, wire an automation, or run an agent from the same app.
