One workspace

Everything, wired together

Six surfaces that share one engine, one tool registry, and one set of safety rules — so an automation can start an agent, a chart can pin to a dashboard, and chat can drive any of them.

Chat

Stream from any model you connect, with optional retrieval over your own documents — and a command surface that can run anything else in the app.

Data

Connect your PostgreSQL databases, build document collections for retrieval, and browse tables read-only. Your data never leaves your machine.

Dashboards

Describe a dashboard; the model writes the SQL and picks the charts. Orrery saves it as a spec and refreshes it for free — the AI is the designer, not the renderer.

Automations

Visual, fixed-recipe workflows on a canvas — triggered by a schedule, a webhook, a manual run, or a database change. You design the steps.

Agents

Goal-driven workers that plan, act, self-check, and improve within a strict scope — keeping learning notes and handing off to automations or to you.

Media Hub

A playground for image and video generation on your own media keys or a local model — prompts and settings saved, assets reusable everywhere.

🔒 Security is the floorKeys live in your OS keychain. Model-written code runs only in a locked-down sandbox. AI-written SQL is shown, not hidden.
🛰️ Local-firstThe backend binds to localhost. No analytics, no crash reporting, no phone-home. Only the providers you chose are ever contacted.
🎯 Accuracy over guessingQueries are parameterized and validated. What the AI can't ground in your schema or data, it declines instead of inventing.
🧩 Bring your own everythingYour model accounts, your API keys, your database. Orrery is the framework that ties them together — you stay in control.

Getting started

From zip to working in four steps

Extract

Unzip the whole folder and keep it together — run setup-orrery.bat, not the loose exe.

Pick a database

Use the bundled Docker PostgreSQL, or paste your own connection string. pgvector powers retrieval.

Connect a model

Add an API key or connect a plan account in Settings. Keys are stored in your OS keychain.

Start working

Chat, build a dashboard, wire an automation, or set an agent loose — from one window.
