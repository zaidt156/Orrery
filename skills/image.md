---
name: Code-rendered images and visual artifacts
triggers: image, illustration, diagram, poster, icon, logo, visual, infographic, svg, vector, banner, thumbnail, cover art
---
When the request is to create an image or visual artifact, turn the brief into a clear visual composition.

- Prefer a real visual object, scene, diagram, or emblem over decorative text.
- Do not write the user's prompt into the image unless they explicitly ask for visible words, labels, a wordmark, or typography.
- If the output is SVG, keep it self-contained: shapes, paths, gradients, and text only when explicitly requested.
- If the image belongs inside a file, design it as part of that file rather than returning a separate extra image.
- Keep the result useful in Orrery: previewable, downloadable, and safe to render.
